# Demo Resources

**Demo Resources** is a single place for Salesforce Solutions Engineers to find guided demo scenarios, ready-to-deploy demo tools, and Cursor enablement. Everything is self-contained in this repo: scenario prompts, deployable utilities, Cursor setup guide, and step-by-step instructions to build and deploy solutions to your org.

---

## What's in this repo

| Resource | Description |
|----------|-------------|
| **[Demo Scenarios](./Demo%20Scenarios/)** | Industry-specific scenarios (Energy, Aviation, Agriculture, Nonprofit, etc.) with copy/paste Cursor Agent prompts (Steps 01–07). Step 07 (push to GitHub) is **optional**. |
| **[Demo Tools](./Demo%20Tools/)** | Ready-to-deploy utilities for generating realistic demo data and content (AI Case Generator, etc.). Deploy once, use repeatedly. |
| **[Cursor Enablement](./Demo%20Scenarios/Cursor%20Enablement/)** | Beginner guide: install Cursor, set up Salesforce CLI, run Step 00, then Steps 01–07 in Cursor Agent, and deploy to your org. |

---

## Quick start

### Demo Scenarios
1. Open **[Demo Scenarios](./Demo%20Scenarios/)** and pick a scenario (e.g. [PawPrint Rescue Network](./Demo%20Scenarios/Nonprofit%20%26%20Rescue/PawPrint%20Rescue%20Network/) for beginners).
2. Read **[Cursor Enablement](./Demo%20Scenarios/Cursor%20Enablement/README.md)** and follow Step 00 in your terminal, then open the project in Cursor.
3. Copy/paste each step prompt (01–06 required; 07 optional) into Cursor Agent and deploy to your scratch org or sandbox.

### Demo Tools
1. Open **[Demo Tools](./Demo%20Tools/)** and pick a tool.
2. Read the tool's README for prerequisites and deploy instructions.
3. Deploy directly to your org with `sf project deploy start`.
